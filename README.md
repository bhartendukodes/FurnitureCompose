# FurnitureCompose - Furniture Shopping UI

A modern furniture shopping app UI built entirely with Jetpack Compose, showcasing Material Design 3 components and clean architecture.

## Features

- Beautiful furniture catalog UI
- Product detail screens
- Shopping cart interface
- Smooth animations with Compose
- Material Design 3 theming
- Responsive layouts

## Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Design:** Material Design 3
- **Architecture:** MVVM
- **Navigation:** Compose Navigation

## Getting Started

1. Clone the repository
```
git clone https://github.com/bhartendukodes/FurnitureCompose.git
```
2. Open in Android Studio (Arctic Fox or later)
3. Build and run
